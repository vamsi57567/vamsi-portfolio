03.5   Deep Case Study

TRUSTWORTHY AI
LLM HALLUCINATION DETECTION

8-layer RAG pipeline to detect and reduce LLM hallucinations — evaluated on 15,000 FEVER benchmark claims.

The Problem
LLMs confidently state incorrect facts. Standard single-retrieval RAG systems miss ambiguous claims and return false-confident answers with no uncertainty signal.
My Approach
Combined dense (FAISS) and sparse (BM25) retrieval in an 8-layer pipeline, then added a DeBERTa NLI verification layer to fact-check generated answers against retrieved evidence.
The Result
+5.95pp accuracy improvement over baseline on 15,000 FEVER benchmark claims. Significantly better uncertainty recognition for ambiguous queries.
